Education Management System – Deep Dive

Dynamic education discovery platform built with the MERN stack and ML-assisted recommendations.

Key Capabilities

  • Smart search & comparison: Users explore institutes and courses using filters like city, fee range, specialization, mode of study, and rating bands.
  • AI-driven recommendations: Personalized suggestions based on career goals, preferred locations, and budget constraints.
  • Instant-query chatbot: ML-inspired bot that retrieves institute profile, fee structure, and nearby options without manual navigation.

Tech & Design Highlights

  • MERN stack for a responsive, SPA-style user experience.
  • Structured data models for institutes, courses, fees & reviews.
  • Filtering & ranking logic for transparent, career-aligned choices.
  • Analytics layer to monitor search trends and user behavior.

This demo focuses on the front-end. The full version connects to real APIs and ML services for live recommendations and analytics.

Healthcare Assistance Platform – Deep Dive

Full-stack microservices system that supports patients from query to appointment, powered by ML chatbots.

Key Capabilities

  • Symptom-aware ML chatbot: Guides patients to relevant departments or specialists based on initial complaints.
  • Automated patient support: Helps with appointment guidance, hospital information, and basic FAQs without manual calls.
  • Operational efficiency: Reduces manual intervention, especially during peak hospital hours.

Tech & Architecture

  • Java Spring Boot microservices for backend APIs.
  • Angular SPA for responsive, modular front-end UI.
  • MySQL for structured storage of hospital & patient data.
  • Python-based ML chatbot service integrated with core APIs.
  • Deployed using Docker & Kubernetes for scale and reliability.

This front-end demo mirrors how the production system behaves, highlighting architecture, flows, and patient-centric design.
